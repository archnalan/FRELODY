# The song editor (Compose)

> Learn what the Compose editor does and how its SongBoard lays out your song so you can build a clean, playable chart.

## Overview

**Compose** is FRELODY's song editor. It opens at `/compose` with the heading **Song Editor**, and it's where you build a song from scratch or polish one you've imported. If you'd rather start from an existing recording, you can import a song first and tidy it here.

At the top sits the song form: a **Song #**, the **Song Title**, an original key and a working key, a transpose delta button to shift between them, a **Book/Artist** picker, and a **Save Song** button. Set these once and the rest of the editor lines up around them.

Below the form, your song is laid out on the SongBoard as sections, rows and segments, organised into tabbed section boards. This is where lyrics and chords come together, line by line, so the chart reads the way you'd play it.

Compose is a member feature that needs the org **Editor** or **Contributor** role. If you can't open it, ask your organisation owner to grant one of those roles.

## Tips

- Set the **Song Title** and keys before you start editing — the transpose button and chord display follow them.
- Work in small steps and hit **Save Song** often; Compose also keeps an unsaved-session backup for you.
