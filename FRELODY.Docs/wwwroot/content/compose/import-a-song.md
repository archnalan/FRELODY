# Import a song

> Bring an existing song into Compose by pasting text, dropping a URL, uploading a file or searching the library — FRELODY structures it for the editor.

## Overview

The **Import song** panel saves you typing a whole chart by hand. Its subtitle says it plainly: "Paste, scan, upload or search — we'll structure it for the editor." You give it some source material and FRELODY works out the sections, lines and chords.

It accepts four kinds of input: lyrics-with-chords pasted as text, a pasted song **URL** (FRELODY scrapes the page), an uploaded file (read with on-image OCR), or a search of the FRELODY library. The single field shows the prompt "Paste lyrics with chords, drop a URL, upload a file, or search the library…".

Once FRELODY has extracted the song, it stages the result as **Ready to apply** and shows you the section, line and chord counts so you can sanity-check it before committing. You then decide how it lands in the editor.

This is a member feature inside Compose. After importing, you'll fine-tune the result on the SongBoard.

## Steps

1. In Compose, open the **Import song** panel.
2. Choose your source: paste lyrics with chords, paste a URL, upload a file, or search the library.
3. Wait for FRELODY to extract the structure — the staged result shows as **Ready to apply** with section, line and chord counts.
4. Choose **Replace** to swap in the imported song, or **Append** to add it to what's already in the editor.
5. Review the result on the SongBoard and edit anything that needs a tidy.

## Tips

- Pasted text works best when chords sit on their own line above the lyrics — the way most chord sheets are written.
- Use **Append** when you're stitching parts together; use **Replace** when you're starting a fresh chart.
- Always glance at the section, line and chord counts before applying — they're a quick check that the import read the song correctly.
