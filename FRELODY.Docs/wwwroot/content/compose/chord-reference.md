# The chord reference

> Browse FRELODY's chord library at `/chords`, search for a chord and open any one for its full details.

## Overview

`/chords` is FRELODY's chord reference. The page is headed "Chords" and shows a grid of chord cards, each with its charts, so you can see shapes at a glance. It's a public page — anyone can browse it.

To find a specific chord, use the **Search a chord...** box at the top. There's also an **Add Chord** button that opens a create dialog when you want to add one that's missing.

Open any chord to reach its detail page, headed "Chord Details" with the line "View and manage chord information". From there you'll find an **Edit Chord** button and a back button to return to the list.

## Steps

1. Open `/chords`.
2. Type in the **Search a chord...** box to filter the grid.
3. Select a chord card to open its **Chord Details** page.
4. Use **Edit Chord** to change it, or the back button to return to the list.
5. Use **Add Chord** if you need to create a chord that isn't there yet.

## Tips

- The search box filters as you type — start with the root note to narrow things down quickly.
- The reference pairs naturally with the editor: find a shape here, then use it when adding chords to a song.
