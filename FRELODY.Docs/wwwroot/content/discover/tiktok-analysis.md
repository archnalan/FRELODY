# Analyse a TikTok video

> Paste a TikTok link to get its chords in a vertical, play-along view.

## Overview

Heard a sound on TikTok you want to learn? Paste the video link and FRELODY analyses it just like a YouTube song. You reach the TikTok analysis page by pasting a `tiktok.com/@user/video/…` link in **Discover**, which opens `/discover/tiktok` with your link attached.

The analysis pipeline and the playback experience are the same as YouTube: FRELODY detects the chords, syncs them to the audio, and gives you the chord chart, beat strip and meta chips. The difference is the player. Because TikTok videos are portrait, the floating player runs vertically in a 9:16 frame to match.

One thing to note: TikTok analyses aren't saved to your library, so there's no **Save to Library** option here. You can still come back to it the same day from [Today's Songs](/docs/discover/todays-songs). An **Open on TikTok** link takes you to the original video whenever you want it.

## Steps

1. Copy the link to the TikTok video you want to learn.
2. Paste it into the **Discover** search box. FRELODY routes it to the TikTok lane and opens `/discover/tiktok`.
3. Start the analysis and wait while FRELODY detects the chords.
4. Play along using the vertical player and the synced chord chart.
5. Tap **Open on TikTok** any time to view the original video.

## Tips

- TikTok transcriptions aren't persisted, so save isn't available. Find the same song again later in [Today's Songs](/docs/discover/todays-songs) within the replay window.
- The same free daily limits and length caps apply. See [Free daily limits and song length](/docs/discover/free-limits).
