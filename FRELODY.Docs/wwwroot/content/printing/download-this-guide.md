# Download this guide

> Save any page of this documentation — or the whole guide — as a PDF to read offline.

## Overview

This page is about the documentation site itself, not the FRELODY app. If you'd like a copy of these help pages to read offline, print, or keep for reference, you can save them as a PDF straight from your browser.

Every page in this guide is built to print cleanly: navigation, buttons and any embedded videos are hidden, so you get just the words and pictures. You can download a single page or the full guide.

## Steps

1. Open the page you want to keep (or the guide's contents page for the whole guide).
2. Open your browser's print command — usually **Ctrl + P** on Windows or **Cmd + P** on Mac.
3. Set the destination to "Save as PDF".
4. Check the preview, then save the file to your device.

## Tips

- Because videos are hidden in print, every walkthrough on this site is also summarised in text — so nothing is lost in your PDF.
- To keep links clickable, choose a PDF destination rather than printing to a physical printer.
- For the complete guide in one file, start from the top-level contents page before you print.
