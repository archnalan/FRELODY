# Export songs as PDF

> Save a clean chord sheet as a PDF, a Premium feature in FRELODY.

## Overview

A PDF is the easiest way to keep a song you can read anywhere, e-mail to a bandmate, or print later. **PDF export** is one of FRELODY's Premium features — its plan-card description is "Export clean chord sheets as PDF".

To be straight with you: FRELODY doesn't have a separate, standalone "Export PDF" page in the app today. PDF export is offered as a Premium capability and reached through the same print path as a chord chart. The most reliable way to get a PDF right now is to print a song and choose "Save as PDF" as the destination.

If you're on the free plan, see [Plans and billing](/docs/pricing/plans) to learn what Premium unlocks.

## Steps

1. Open the song you want from your library.
2. Open the song's **Share** dropdown.
3. Choose **Print**.
4. When your device's print dialog opens, set the destination to "Save as PDF" (the wording varies by browser).
5. Save the file to your device.

## Tips

- The look of the PDF depends on the song display mode you have set — choose lyrics-and-chords for a compact sheet.
- If printing says it's unavailable on your device, try a desktop browser, where "Save as PDF" is usually available.
- Keep your exported PDFs in a folder per songbook so they're easy to find before a rehearsal.
