# Tenants

> Find and review the application tenants from the tenant management screen.

## Overview

The tenants screen is where you manage application tenants. Open `/admin/tenants`, titled "Tenant Management — Manage application tenants."

The page shows a searchable, virtualised list of tenants. Use the **Search tenants...** box to filter; a clear button resets your search. If something goes wrong loading the list, an error alert explains the problem.

Use this screen when you need to locate a specific tenant or scan the list of tenants in the application.

## Steps

1. Open `/admin/tenants`.
2. Type in the **Search tenants...** box to filter the list.
3. Use the clear button to reset your search and see all tenants again.

## Tips

- The list is virtualised, so it stays responsive even with many tenants.
- If an error alert appears, retry after the underlying issue is resolved.
